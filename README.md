# React Pokédex

## Introduction

React Pokédex is a mobile-first responsive web-based UI for those wanting to view a lists of Pokémons in numerical order, which you are able to traverse throgh by using pagination buttons. 

## Screenshot

![Screenshot](https://raw.githubusercontent.com/jsubroto/react-pokedex/main/sceenshot.png)

## Technologies

React Pokédex was built using the following technologies:

* [React](https://reactjs.org/) (via [`create-react-app`](https://github.com/facebookincubator/create-react-app))
* [TypeScript](https://www.typescriptlang.org/)
* [Tailwind CSS](https://tailwindcss.com/)
* [PokéAPI](https://pokeapi.co/)
* [Fetch API](https://developer.mozilla.org/en-US/docs/Web/API/Fetch_API/)
